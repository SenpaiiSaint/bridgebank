# FinLink Security

## Core Security Features

### Password Protection
- bcrypt encryption
- Requirements:
  - 8+ characters
  - Numbers and letters
  - Uppercase and lowercase
  - Special characters

### Session Security
- 30-minute timeout
- Secure cookies

### Rate Limiting
- Request limits on sensitive endpoints
- 1-minute cooldown

### Data Protection
- Encrypted sensitive data
- Input sanitization
